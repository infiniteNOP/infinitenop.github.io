Welcome to my portfolio page!

This website, written in a somewhat crude way using HTML, serves as a list of all of my projects. Feel free to look around, and, maybe, learn something along the way, as I did while creating these projects.

For your convenience, the projects are organized in the following categories:

  1. Hardware
  2. Software

Hardare projects

These projects are either pure hardware designs, or they are written in an Hardware Description Language (HDL), like Verilog.

  1. nopCPU

    This is a CPU I designed in verilog, in order to acquaint myself with the workings of such devices. It has been tested on a Spartan3 FPGA board, and it seems to work fine. However, it has not been used in a serious/large project, and, as such it remains largely untested. Please do take the no warranty warning seriously.

    This project was featured on hackaday

  2. ntsc_gen

    A simple NTSC generator. Not tested as of the time of writing.

You can see some other hardware designs of mine, including an 6502 computer which does not use any other 65XX family chips besides the processor, by visiting my old homepage

Software projects

These projects are supposed to be executed on a typical computer system, like the one you are using to read this very page.

  1. nopCPU_assembler

    An assembler for nopCPU (see Hardware section). I do not think it is in working condition, so feel free to fix it, or write a better assembler, if you feel like doing it. In the meantime, you can wait until I decide I want to write an operating system for my CPU design, and be forced to fix the assembler as a result. Written in TCL.

  2. CPUemu

    A toy CPU emulator I wrote. Its conception was a precursor to me designing my Verilog CPU (see hardware section). Written in C.

  3. infiniteBBS

    A BBS thing I once wrote. It was never used for anything, uses PostgresSQL, is written in C, and has been abandoned for a long time. It does work, though.
